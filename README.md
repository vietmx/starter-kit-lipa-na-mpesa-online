### Lipa na Mpesa online tutorial.

Steps lipa na mpesa online using Node.js

- Clone the repository.
- Run the following command:

```bash
npm install
```

- Go through the `index.md` file.

- Enter your credentials on the `.env` file.

- Start the server:

```bash
npm run dev
```

- Start Ngrok

```bash
npm run ngrok
```
